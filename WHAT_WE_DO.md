# What We Do

This document will explain what QDE Systems Group members may work on together.

- deterministic execution infrastructure operation;
- QDE setup and validation workflow discussion;
- IBKR / TWS / IB Gateway technical validation discussion;
- audit and log review patterns;
- bug reports and reproducible issue reports;
- AI-agent workflow and prompt improvement;
- configuration discipline and change review;
- sanitized validation metrics;
- model / promotion / runtime behavior discussion at a technical level.

The group is for technical operation and infrastructure review, not trading
signals or managed trading.
