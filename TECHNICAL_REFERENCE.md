# Technical Reference

This document will link public QDE-Systems technical reference material.

Primary public reference:

- [qde-systems-infrastructure](https://github.com/Stefan-Len/qde-systems-infrastructure)

That repository is a public architecture sample for deterministic execution
infrastructure: risk-gated decisions, deterministic order planning, audit trails,
and fail-closed execution boundaries.

It is not the commercial QDE source package and does not include private QDE
source code, broker integrations, IBKR adapter code, credentials, private market
data configuration, model artifacts, or buyer documentation.
